# SmartCalc Mobile

Android-first Flutter-приложение для теплотехнических расчетов ограждающих конструкций. Проект опирается на пользовательские сценарии `smartcalc.ru`, но не копирует сайт, его код или данные.

## Текущее состояние

- Инициализирован Flutter-проект и каркас модульной архитектуры.
- Зафиксированы ТЗ, план реализации, правила разработки, архитектурное решение и чек-лист.
- Добавлены seed-данные по климату, материалам и нормативным ссылкам.
- Реализован нормативный экран `thermocalc`: расчет сопротивления теплопередаче, температурный профиль по слоям, MVP-скрининг влагорежима и ссылки на примененные нормы.
- Добавлены эталонные reference cases для расчета теплозащиты и влагорежима в тестах.
- `drift`-хранилище и `PDF`-генерация еще не реализованы.

## Быстрый старт

```bash
flutter pub get
```

## Замечание по точности

Текущий экран расчета закрывает теплозащиту `v1` и MVP-скрининг влагорежима. Полный расчет влагонакопления, сезонные режимы и производственный `PDF`-отчет пока не реализованы, поэтому результат нужно трактовать как инженерный экран предварительной проверки, а не как исчерпывающий нормативный отчет.
