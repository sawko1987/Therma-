# SmartCalc Mobile

Android-first Flutter-приложение для теплотехнических расчетов объекта, его помещений и ограждающих конструкций. Проект опирается на пользовательские сценарии `smartcalc.ru`, но не копирует сайт, его код или данные.

## Текущее состояние

- Инициализирован Flutter-проект и каркас модульной архитектуры.
- Зафиксированы ТЗ, план реализации, правила разработки, архитектурные решения и чек-лист.
- Добавлены seed-данные по климату, материалам и нормативным ссылкам.
- Реализован `wizard` шага 2: объект, комнаты, поверхности/проемы и расчет теплопотерь по комнатам и по всему объекту.
- Сохранен отдельный `construction preview` для просмотра характеристик одной конструкции.
- Сезонное потребление для газа, электричества и теплового насоса, локальное хранилище и `PDF`-экспорт еще не реализованы полностью.

## Быстрый старт

```bash
flutter pub get
flutter run
```

## Замечание по точности

Экран `Construction Preview` показывает одну конструкцию, ее слои и preview-индикаторы. Он не должен считаться итоговым расчетом здания. Нормализованный результат для шага 3 формируется только в `wizard` шага 2 через модель объекта, комнат, поверхностей и проемов.
