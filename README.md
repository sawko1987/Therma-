# SmartCalc Mobile

Android-first Flutter-приложение для теплотехнических расчетов ограждающих конструкций. Проект опирается на пользовательские сценарии `smartcalc.ru`, но не копирует сайт, его код или данные.

## Текущее состояние

- Инициализирован Flutter-проект и каркас модульной архитектуры.
- Зафиксированы ТЗ, план реализации, правила разработки, архитектурное решение и чек-лист.
- Добавлены seed-данные по климату, материалам и нормативным ссылкам.
- Собран `preview`-экран `thermocalc` с визуализацией сечения и графиков.
- Нормативный расчетный движок, `drift`-хранилище и `PDF`-генерация еще не реализованы полностью.

## Быстрый старт

```bash
flutter pub get
flutter run
```

Текущий экран расчета показывает архитектурный `preview` и демонстрацию визуализаций. Он не должен считаться нормативным результатом до реализации и верификации полного расчетного движка.
