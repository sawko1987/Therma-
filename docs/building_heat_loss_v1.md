# Расчет теплопотерь здания v1

## Что реализовано

- Отдельный экран расчета теплопотерь здания.
- Единый `building-loss service`, который используется и для детального экрана, и для summary в конструкторе дома.
- Разбивка результата по дому, комнатам и ограждающим элементам.
- Раздельный учет потерь через непрозрачные участки и через проемы.
- Расчет внутренней температуры по типу комнаты `RoomKind` из локального dataset `assets/data/room_kind_conditions.seed.json`.
- Базовый расчет вентиляционных потерь по комнате через `ACH`, объем помещения и расчетный перепад температур.

## Что входит в v1

- Потери через непрозрачные ограждения по формуле `Q = ΔT / R * A`.
- Потери через окна и двери по формуле `Q = ΔT * U * A`.
- Потери на вентиляцию по формуле `Q = 0.335 * ACH * V * ΔT`.
- Баланс покрытия отопительными приборами по комнате и по дому.

## Что пока не входит

- Инфильтрация.
- Мостики холода.
- Коэффициенты запаса и расширенный подбор отопления.
