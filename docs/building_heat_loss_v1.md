# Расчет теплопотерь здания v1

## Что реализовано

- Отдельный экран расчета теплопотерь здания.
- Единый `building-loss service`, который используется и для детального экрана, и для summary в конструкторе дома.
- Разбивка результата по дому, комнатам и ограждающим элементам.
- Раздельный учет потерь через непрозрачные участки и через проемы.
- Расчет внутренней температуры по типу комнаты `RoomKind` из локального dataset `assets/data/room_kind_conditions.seed.json`.
- Расчет вентиляционных потерь через отдельный persisted-модуль `ventilationSettings`.
- Расчет инфильтрационных потерь через внешние проемы по preset-герметичности окна или двери.
- Расчет внутреннего теплообмена между смежными комнатами через общую внутреннюю перегородку с room-level `net heating demand`.
- Отдельный учет линейных мостиков холода через `ψ`.

## Что входит в v1

- Потери через непрозрачные ограждения по формуле `Q = ΔT / R * A`.
- Потери через окна и двери по формуле `Q = ΔT * U * A`.
- Потери на вентиляцию по формуле `Q = 0.278 * n * V * ρ * c * ΔT * (1 - η)`.
- Потери на инфильтрацию по формуле `Q = 0.335 * L * ΔT`, где `L` собирается по площади проемов и их preset-герметичности.
- Потери на линейные мостики по формуле `Q = ψ * L * ΔT`.
- Перетекание тепла между смежными комнатами по формуле `Q = ΔT / R * A` для общей внутренней перегородки.
- Баланс покрытия отопительными приборами по комнате и по дому.

## Что пока не входит

- Коэффициенты запаса и расширенный подбор отопления.
