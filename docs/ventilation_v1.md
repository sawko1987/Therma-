# Вентиляция v1

## Что реализовано

- Persisted-модель `ventilationSettings` в `Project`.
- Отдельный экран `Вентиляция` на `Dashboard`.
- Отдельный сервис `VentilationHeatLossService` вне widget-дерева.
- Интеграция вентиляционных потерь в `building heat loss` и `heating economics`.
- Поддержка сценариев `natural`, `forced`, `heatRecovery`.

## Что входит в v1

- Настройка воздухообмена для комнаты или всего дома.
- Расчет по формуле `Q_vent = 0.278 * n * V * ρ * c * ΔT * (1 - η)`.
- Константы воздуха вынесены на уровень файла сервиса.
- Учет КПД рекуперации только для `heatRecovery`.
- Разнесение общедомовой вентиляции по комнатам пропорционально объему.

## Нормативная ссылка

- `СП 60.13330.2020` — раздел по вентиляции и расчету теплопотерь приточного воздуха.
