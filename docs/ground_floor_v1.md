# Полы по грунту v1

## Что реализовано

- Отдельный экран `Полы по грунту` на `Dashboard`.
- Persisted-модель `groundFloorCalculations` внутри versioned `Project`.
- Auto-migration legacy-проектов к новому формату без обратного слома.
- Отдельный сервис расчета поверх существующего нормативного термодвижка.

## Что входит в v1

- Один поддержанный сценарий: `плита по грунту`.
- Переиспользование слоистой конструкции типа `Пол`.
- Дополнительное эквивалентное сопротивление грунта и утепленной кромки.
- Проверка итогового `R` против нормативного требования для пола.
- Расчет теплопотерь через пол по грунту по площади и расчетной `ΔT`.
- Хранение нескольких расчетов в проекте и выбор активного расчета.

## Что пока не входит

- Пол по грунту на ленте.
- Плита над подвалом.
- Привязка расчета к `houseModel` floor-элементам.
- `PDF`-экспорт или отдельный отчет.

## Принятая модель v1

- `R конструкции` берется из существующего расчета слоистой конструкции.
- `R грунта` считается как добавочное эквивалентное сопротивление по геометрии плиты и параметрам утепления кромки.
- Итоговое сопротивление считается как сумма `R конструкции` и `R грунта`.
- Теплопотери считаются по формуле `Q = ΔT / R_total * A`.

## Нормативные ссылки

- `СП 50.13330.2012` — базовая теплозащита ограждающих конструкций.
- `СП 131.13330.2020` — климатические параметры.
- `ГОСТ Р 54851-2011` — приведенное сопротивление теплопередаче.
